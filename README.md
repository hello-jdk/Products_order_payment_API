
# 회원 정보 & 상품 장바구니,주문,결제 관리 API 서버

**22.09.10 ~ 22.09.14**

## 개요

회원관리, 상품관리, 결재관리, 주문내역 관리에 대한 RESTful API 서버

- 등급에 따라 일반사용자, 관리자가 나누어집니다.
- 일반사용자는 상품에 대한 CRUD를 제외한 회원관리, 장바구니, 주문, 결제에 대한 기능을 사용가능합니다.
- `상품에대한 장바구니 담기` -> `장바구니에 있는 상품에대한 주문` -> `결재` 순 으로 이루어집니다.
